# 🔐 Random Password Generator

A simple and responsive Random Password Generator built using **HTML, CSS, and JavaScript**. Generate strong and secure passwords instantly with a clean and modern user interface.

## 🚀 Live Demo

🔗 https://abhinavvv12.github.io/Random-Password-Generator/

## 📌 Features

- Generate random secure passwords
- Clean and modern UI
- Responsive design
- One-click password generation
- Fast and lightweight
- Beginner-friendly project

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 How It Works

1. Open the application.
2. Click the **Generate Password** button.
3. A random password is generated instantly.
4. Use the generated password wherever needed.
